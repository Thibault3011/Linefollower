## specifications
<br />
microcontroller: Arduino Leonardo
<br />
motors: Micro metal gearmotor HPCB 6V
<br />
h-bridge: DRV8833
<br />
sensors: QTR-8A
<br />
batteries: 18650 7,4V - 1,3A
<br />
wireless communication: HC-05
<br />
distance sensor - motors: 11 cm
<br />
weight:/
<br />
speed: 0.3
